# Nick Peril | Personal Portfolio Website

Personal portfolio website for **Nick Peril**, a career changer moving from sales, inventory, and business operations into IT with a primary focus on **data analytics**.

This site is designed to help recruiters and hiring managers quickly understand my story, my technical direction, and the projects I have built across data analytics, dashboarding, machine learning, AWS cloud, and software development.

## Portfolio Focus

I am positioning myself for data analyst roles while staying open to opportunities that connect business operations with technical problem solving.

Core areas highlighted in this portfolio:

- **Data Analytics:** Python, SQL, exploratory data analysis, business insight generation
- **Dashboarding:** Tableau and visual storytelling for decision support
- **Machine Learning:** supervised learning, recommendation systems, model evaluation
- **Cloud and DevOps:** AWS labs, infrastructure concepts, deployment workflows
- **Software and Web Development:** practical business-focused web projects
- **Events:** hackathons, developer events, and community learning experiences
